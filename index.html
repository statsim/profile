Data profiling online

Profile data and generate visual summaries of your CSV datasets using this free and open-source web app

Understanding data is a crucial skill in many industries. Generally, people are not good at recognizing patterns and extracting insights from raw tabular datasets. We are not computers. That's why data visualization and profiling are so important and used so often. They convert numbers into something regular people can understand. Charts. Trends. Statistics. Data profiling makes it possible to generate bird-view summaries of tabular files. It provides detailed information and descriptive statistics about each variable stored in a dataset. StatSim Profile is a free and open-source data profiling tool working in a browser. It processes files locally, without loading them to a web server, and works even with big datasets. Up to gigabytes!

Open large CSV files

The app uses online statistical algorithms to process data sequentially without loading it all to memory. To make it possible, we developed and open-sourced OnlineStats.js, a collection of JavaScript modules for online statistical analysis.

Understand your data

StatSim Profile includes various statistical methods that can help you understand data better. It generates a bird-view summary of a CSV dataset, including variable types, missing values, descriptive statistics, histograms, top values, first five records.

100% free and client-side

StatSim Profile is a JavaScript web app living in your browser. It doesn't have any server-side code, so any dataset you profile stays on your machine. It's also fully open-source and hosted using GitHub Pages. You can even download and use it locally.
